What is Essential Pascal?

"Essential Pascal" is a free online introduction to the Pascal programming language. This book has been written by Marco Cantù and covers the language form the perspective of Borland's Delphi development environment. You can find more information about this book in its Introduction.

Getting Essential Pascal

There are different ways of getting the latest version of Essential Pascal. You can:

  • See the version available on Delphi 5 Companion CD (which is still quite up-to-date)
  • Browse the book online, here (see below for the online version in other languages)
  • Download the file epascal_v1.zip (241 KB) with the complete text and figures (see below for the download version in other languages)

In any case, remember also to download the zip file with the complete source code of the examples discussed in the book.
